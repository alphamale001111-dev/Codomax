# 🐍 Python Basics Learning Guide (Day 2)

Welcome to the Python Basics Learning Course! This project is designed to help you learn Python's foundational concepts from scratch through hands-on code examples, explanations, and interactive terminal lessons.

---

## 📚 Curriculum & Course Structure

This course consists of 4 interactive conceptual lessons and 3 simple practical applications. Each script is fully annotated with comments and contains a **mini-quiz** or **game** at the end that runs directly in your terminal.

### 📝 Lessons

1. **Variables & Data Types**
   - **Topics**: Creating variables, dynamic typing, core types (int, float, str, bool, list, tuple, dict, set), and type casting.
   - **Interactive File**: [variables_data_types.py](file:///Users/d.k./Desktop/Codomax%20internship/Day%202/variables_data_types.py)
   - **Run Command**: 
     ```bash
     python3 "Day 2/variables_data_types.py"
     ```

2. **Python Operators**
   - **Topics**: Arithmetic (`+`, `-`, `*`, `/`, `//`, `%`, `**`), Assignment (`+=`, `-=`), Comparison (`==`, `!=`, `<`), Logical (`and`, `or`, `not`), and Membership (`in`, `not in`).
   - **Interactive File**: [operators.py](file:///Users/d.k./Desktop/Codomax%20internship/Day%202/operators.py)
   - **Run Command**: 
     ```bash
     python3 "Day 2/operators.py"
     ```

3. **Conditionals & Loops**
   - **Topics**: Conditional branches (`if`, `elif`, `else`), `for` loops, `while` loops, the `range()` function, and loop control (`break`, `continue`, `pass`).
   - **Interactive File**: [loops.py](file:///Users/d.k./Desktop/Codomax%20internship/Day%202/loops.py)
   - **Run Command**: 
     ```bash
     python3 "Day 2/loops.py"
     ```

4. **Functions**
   - **Topics**: Declaring functions (`def`), parameters and return values, default arguments, variable positional/keyword arguments (`*args`, `**kwargs`), and anonymous `lambda` functions.
   - **Interactive File**: [functions.py](file:///Users/d.k./Desktop/Codomax%20internship/Day%202/functions.py)
   - **Run Command**: 
     ```bash
     python3 "Day 2/functions.py"
     ```

## 🚀 How to Get Started

1. Open your terminal in this workspace.
2. Run any lesson file using `python3`. For example:
   ```bash
   python3 "Day 2/variables_data_types.py"
   ```
3. Read the terminal printout explaining the concepts.
4. Complete the interactive mini-quiz in your terminal.
5. Inspect the file content to see the comments and code implementation details!
